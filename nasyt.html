脚本简介

NAS油条Linux工具箱是一个功能强大的Linux系统管理脚本，由NAS油条开发，该脚本集成了大量实用的Linux系统工具和功能，为用户提供一站式的系统管理和维护解决方案。


本脚本支持多种Linux发行版和Termux终端，包含了系统信息查看、软件包管理、网络工具、Docker管理、各种服务器面板安装、Bot机器人部署、游戏娱乐工具等丰富功能。无论是Linux新手还是资深管理员，都可以通过本工具简化日常系统管理任务。

主要功能

NAS油条Linux工具箱提供了丰富多样的功能模块，涵盖了Linux系统管理的各个方面：

系统信息查看

查看CPU、内存、硬盘使用情况，系统运行时间，进程数量，IP地址，地理位置等系统信息。

网络工具

Ping测试、网络连通性测试、端口扫描、内网穿透、Tmux终端工具等网络相关功能。

软件包管理

支持多种包管理器(apt, yum, dnf, pacman等)，软件安装、卸载、更新、镜像源更换等功能。

服务器面板

一键安装宝塔、AMH、1Panel、MCSManager、小皮、GMSSH、Dpanel等各种服务器管理面板。

Bot机器人

部署Secluded、TRSS、Astrbot、Napcat、OneBot、Easybot、Koishi、MaiBot、Karin等多种机器人。

娱乐游戏

俄罗斯方块、贪吃蛇、太空入侵、黑客帝国屏保、盆栽艺术、可视化音频等终端游戏和娱乐工具。

Docker管理

Docker信息查看、容器管理、镜像管理等Docker相关功能。

其他脚本集成

集成了亚洲云LinuxTool、木空云LinuxTool、TMOE、kejilion、v2ray一键安装等实用脚本。

安装与使用

NAS油条Linux工具箱安装简单，支持一键安装和更新：

bash -c "$(curl -L https://raw.gitcode.com/nasyt/nasyt-linux-tool/raw/master/nasyt_install.sh)"

安装完成后，直接在终端输入以下命令即可启动：

nasyt

如果遇到权限问题，可以尝试以下命令：

chmod +x /usr/bin/nasyt  # 或 chmod +x $PREFIX/bin/nasyt (Termux)

支持的系统：

  • Debian/Ubuntu系列
  • Fedora/RHEL/CentOS系列
  • Arch Linux系列
  • openSUSE系列
  • Alpine/PostmarketOS系统
  • Gentoo/Funtoo系统
  • Termux终端
  • MacOS系统（测试中）

命令参数

NAS油条Linux工具箱支持多种命令行参数，方便快速执行特定功能：

  • nasyt - 正常启动脚本（显示主菜单）
  • nasyt -g 或 nasyt --gx - 快捷更新脚本
  • nasyt -u 或 nasyt --upsource - 快捷更换软件源
  • nasyt -t 或 nasyt --tmux - 快捷进入tmux管理
  • nasyt -s 或 nasyt --skip - 跳过引导直接进入菜单
  • nasyt -v 或 nasyt --version - 显示脚本版本信息
  • nasyt -h 或 nasyt --help - 显示帮助信息
  • nasyt -b 或 nasyt --backup - 快捷备份/恢复脚本

使用示例：

# 查看脚本版本
nasyt --version

# 更新脚本
nasyt -g

# 直接进入工具箱主菜单（跳过引导）
nasyt -s

# 备份脚本
nasyt -b
