# 航天浏览器主页

一个简洁高效的浏览器主页，专为航天爱好者和高效上网用户设计。

## ✨ 功能特色

### 🔍 智能搜索
- 使用必应搜索引擎作为默认搜索引擎
- 支持回车键快速搜索
- 支持快捷键 (Ctrl+K / Cmd+K) 快速聚焦搜索框

### 🛠️ 工具卡片
1. **文件管理器** - 模拟文件浏览功能
2. **卫星百科** - 快速搜索卫星知识
3. **积满** - 一键跳转到游戏娱乐网站
4. **爱** - 一键跳转到爱心分享网站

### 📁 文件管理器
- 模拟本地文件系统浏览
- 支持文件夹导航
- 显示文件类型和大小
- 响应式设计，适配移动设备

### 🛰 卫星百科搜索
- 弹出式搜索窗口
- 直接跳转到卫星百科网站
- 支持键盘操作

## 🚀 使用指引

### 快速开始
1. 打开 `index.html` 即可使用
2. 在主搜索框输入内容，回车或点击搜索按钮开始搜索
3. 点击工具卡片访问相应功能

### 快捷键
- **Ctrl+K / Cmd+K** - 聚焦搜索框
- **Enter** - 提交搜索
- **ESC** - 关闭所有模态窗口
- **Tab** - 在不同组件间导航

### 文件管理器操作
1. 点击左侧导航栏切换目录
2. 点击文件项查看文件信息
3. 点击文件夹图标进入子目录
4. 点击刷新按钮重新加载文件列表

## ⚠️ 注意事项

1. **文件管理器**：由于浏览器安全限制，当前版本为模拟文件系统。真实文件访问需要用户主动触发文件选择或使用专门的API。

2. **外部链接**：工具卡片会跳转到外部网站，请注意网络安全。

3. **搜索引擎**：默认使用必应，可以根据需要修改 `js/main.js` 中的搜索URL。

4. **本地存储**：本应用不使用本地存储，所有数据在页面刷新后会重置。

## 🔄 自定义配置

### 修改搜索引擎
在 `js/main.js` 文件中修改第52行：
```javascript
const searchUrl = `https://www.bing.com/search?q=${encodeURIComponent(query)}`;
```

### 修改网站链接
在 `js/main.js` 文件中修改相应的跳转链接：
- 第362行：积满网站
- 第369行：爱网站

### 添加新工具
1. 在 `index.html` 中添加新的工具卡片
2. 在 `css/style.css` 中定义样式
3. 在 `js/main.js` 中添加对应的事件处理
